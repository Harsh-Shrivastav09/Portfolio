My Journey
I've always been fascinated by the intersection of creativity and technology, which led me to pursue frontend web development. My journey began with a curiosity to understand how websites work and how visually stunning, functional interfaces are created.

Initially, I immersed myself in learning the foundational building blocks of the web development like HTML, CSS, and JavaScript and some programming Langauges like C and C++. I started by experimenting with small personal projects, like creating simple web pages and playing around with styles and layouts. These early projects helped me grasp the fundamentals and sparked my interest in delivering great user experiences.
